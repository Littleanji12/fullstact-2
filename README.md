My project is a full-stack web application developed using React.js, Node.js, Express.js, and MongoDB. The main purpose of this project is to create a secure and user-friendly platform where users can register, log in, and manage their personal notes or items.

The frontend of the application is built using React.js and designed with a clean card-based interface. The backend is developed using Node.js and Express.js, which handles API requests, user authentication, and database operations. MongoDB is used to store user details and notes securely.

The application includes important features such as user registration, login, JWT authentication, password hashing, and CRUD operations. After logging in, users can create, view, update, and delete their own notes or items. Each user can access only their own data, which improves privacy and security.

This project helped me understand how a real-world web application works, including how the frontend communicates with the backend, how data is stored in a database, and how user authentication is handled securely.
